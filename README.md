C'est quoi ça ?
===============
Spammer est un petit programme écrit en Java qui permet de spammer une touche du clavier ou de la souris de manière extrèmement rapide.
Il peut être considéré comme un bot, et est surtout pratique dans certaines situations pour éviter d'avoir à le faire soi-même. ^^

Je décline toute responsabilité en cas de problème avec le programme. Il est susceptible de consommer beaucoup de ressources processeur en fonctionnement, et vous pourriez vous faire bannir d'un serveur de jeu bêtement si vous l'utilisez pour gagner un avantage...
